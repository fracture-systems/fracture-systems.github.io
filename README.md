# FRACTURE: A window cropping and shading tool created with WebGPU for Linux

This is a tool written in Rust with WebGPU for Gnome Shell to fracture (crop) and shade windows on Linux.

# Install: The recommended way is to use Flathub

[![download-on-flathub](/fracture/img/flatpak-badge/flathub-badge-i-en.png)](https://flathub.org/en/apps/systems.fracture.launcher)
